# 🛒 Grocery List App

A simple and lightweight grocery list application built with vanilla JavaScript. It allows users to add, edit, delete, and persist items using local storage.

## ✨ Features

* Add new grocery items
* Edit existing items
* Delete individual items
* Clear the entire list
* Persistent data using `localStorage`
* Simple alert feedback for user actions
* Clean and minimal UI with Material Icons

## 🚀 How It Works

* Items are stored in the browser using `localStorage`
* Each item is assigned a unique ID using a timestamp
* On page load, saved items are automatically restored
* DOM manipulation is used to dynamically update the list
