# Grocery List App

A simple grocery list app built with HTML, CSS, and JavaScript.

## Features

* Add items
* Edit items
* Delete items
* Clear the full list
* Saves data with `localStorage`

## How to use

Clone the repo and open `index.html` in your browser.

```bash
git clone https://github.com/your-username/grocery-list-app.git
```

## Notes

This project was made to practice DOM manipulation, event handling, and local storage in JavaScript.
